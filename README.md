# xtal syntax definitions for SublimeText
*xtal* is a script language for game development. 

### Features
* Class
* C like syntax
* Dynamically typed
* Easily embedded into C++ applications 
* Closure
* Fiber (a.k.a coroutine or micro thread)
* Byte code VM
* Native thread

See also: https://code.google.com/p/xtal-language/
